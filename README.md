# Number Guessing Game

A simple Python game where the computer picks a random number between 1 and 100 and the user tries to guess it.

## How to play
- Run the program
- Type your guess
- The program tells you if you are too high, too low, or very close
- Keep guessing until you get it right!

## Features
- Random number generation
- Attempt counter
- "Very close" hint when you are within 5 numbers
